ZWS Routes plugin helps you to find your routes.

== Description ==

This small and lightweight plugin is for add and find the routes by day basis. It has two interface, one is for the administrator and the other is for the users. When you install this plugin a table would automatically create to your database named "zws_routes" preceding with the table prefix. Routes are saves in that table.

== Installation ==

1. Upload the entire `zws-routes` folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the `ZWS Routes Menu` to enter your google api key
4. To show ZWS Routes in the frontend, please create a page and use this [zwsroutes-shortcode] shortcode.
